Javascript Language

This week at EDA we got to learn more about Javascript and how to manipulate the DOM. Javascript is comprised of Values, Operators, Expressions, Keywords and Comments. Here are some of the common things you will see when using Javscript:

( ) - Parentheses:

  • Functions - Parenthesis are used when writing and calling Functions. When writing a Function, parenthesis are used for holding parameters, e.g. function perimeter (height, width). When calling a function, parenthesis are used for holding argument values, e.g. perimeter(6,4)
  • Conditional Statements - In conditional statements, parenthesis are used to control the order of operations

[ ] - Brackets:

  • Square Brackets are used to hold values in an array with each value being seperated by a comma. Each value also represents an index within an array. E.g. var numbers = [1, 2 , 3]

{ } - Braces:

  • Curcly Braces are used to notate an object or the opening of a block statement. You can think of the curly braces as the 'Do something' bit of your code. E.g. var myCar = {
    make: 'nissan',
    model: 'pulsar'
    }

'Single' and "Double" Quotes

  • Quotes can be used to surround strings in Javascript. Strings are a collection of letters, numbers and punctuation, and are commonly used in sentences.
    There is no difference between single and double quotes, so they can be used as you will. Just be sure to keep your beginning quote and closing quote the same type. E.g. "This is a sentence"
