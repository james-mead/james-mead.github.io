HTML, CSS and the DOM

This week we got to learn more about HTML, CSS and the Document Object Model. We have been asked to come up with an analogy which best describes what each does and the relationship between them.

I like the anology of building a house. If the Document Object Model is essentially the framework for how a house should be built, HTML is the tools used for putting the house together, and CSS is used to style the house how you would like it.

Boxifying Content

Boxifying content is essentially the pracise of breaking down each element on a webpage into a series of logical "boxes". By boxifying each element, it makes designing a webpage much easier.

The Box Model
