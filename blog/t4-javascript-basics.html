Javascipt Basics

This week at EDA we got to learn about Javascipt. So what is Javascript?

Web Technologies

There are three main technologies to create web pages.

  1. HTML - Which is used for structuring content
  2. CSS - Which is used to style elements on a page
  3. Javascript - Which is used for for manipulating elements based upon events, (like clicking on something, or changing a value in a form)

Control Flow and Loops

Control Flow is an important aspect of programming languages that allow your code to take certain actions based on a variety of scenarios. A great example of the use of control flow is a 'while loop' statement whereby it loops through a block of code as long as a specified condition is true.

while (i < 10) {
text += "The number is " + i;
i++;
}

Accessing Data from Arrays vs Objects

An Array is a special type of variable which can hold more than one value at a time.

var cars = ["Saab", "Volvo", "BMW"];

You can refer to an element in an array by referencing its index number.

cars[0]; // returns the first element in the array, or 'Saab'

Objects can be compared to objects in real life and contain various properties which help describe it. If we are to use the real life object of a car, we know it has several properties such as make, model, year and colour.

var car = {make:"Fiat", model:"500", year:"1990", color:"white"};

You can access object properties in two ways:
objectName.propertyName or objectName["propertyName"]

Functions

A Function is a block of code designed to perform a particular task and is executed once it has been called or invoked. There are several benefits of using Functions within software development, such as the ability to call a Function multiple times to avoid code duplication and minimize errors.
