What is Scope?

How Scope Works in Javascript

In Javascript, scope refers to where variables and functions are accessible, and in what context they are being executed. Code can be defined as having either a global scope or a local scope.

Global scope is the highest level of scope and is the default scope or namespace when writing a new Javascript document. There can only be one Global scope and within it, variables and functions are accessible to an entire document. For example:

// global scope - accessible to the entire document
var firstName = "James";

Local Scope refers to code that is defined and accessbile from within certain parts of your document. Local scopes are created within functions. Any code that is written within a function's block statement is limited to a local scope. For example:

// code within the block statement is limited to a local scope function name () {
var firstName = "James";
var lastName = "Mead";
}
